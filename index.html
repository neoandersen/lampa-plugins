AI Search User API v1.1.0

Умный AI-ассистент для поиска фильмов и сериалов. Работает через известных провайдеров ИИ-моделей таких как OpenAI, DeepSeek, OpenRouter, Together, Groq, Gemini или Custom (включая бесплатные модели) напрямую в обход Lampac. Интегрирована подгрузка постеров и метаданных через OMDb API. Это облегчает получение APi ключа, так как последнее время не проходит регистрация в TMDB.
